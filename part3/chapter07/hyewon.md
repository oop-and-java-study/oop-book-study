# Chapter 07

## 디자인 패턴 개요

> 이전과 비슷한 상황에서 사용했던 설계를 재사용해야하는 경우 해당 설계를 패턴화한것

### 디자인 패턴 이점

- 상황에 맞는 올바른 설계를 더 빠르게 적용할 수 있음
- 각 패턴의 장단점을 통해 설계를 선택하는데 도움을 받음
- 설계 패턴에 이름을 붙임으로써 시스템의 문서화, 이해, 유지보수에 도움을줌

### GoF 디자인패턴

> 객체의 생성, 기능의 확장, 기능의 변경, 구조 등과 관련된 설계의 패턴
